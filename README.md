# Torneio Code War


##REGRAS

Cada jogador só poderá submeter 1 programa.

Você poderá mudar seu programa até a data de encerrramento das inscrições.

Os programas serão divididos em grupos de N participantes cada.

Cada programa de um grupo enfrentará todos os adversários no mesmo grupo.

Os primeiros colocados passarão para a fase seguinte, participando de outros grupos, com outros classificados.

O número de grupos e de fases dependerá do número de participantes no encerramento das inscrições.

Após a execução de cada fase do torneio os participantes poderão rever todas as partidas que desejarem.​

##como funciona?

Aprender →  Submeter → Competir →  Rever
Aprender jogando.

Cadastre-se e evolua seus bots até o Nível 5 com missões guiadas e dicas. Treine à vontade no ambiente de prática.

##Preparar a submissão.

Leve seus bots para a Arena, teste contra outros participantes e ajuste a estratégia. Publique a melhor versão e entre no Ranking.

Inscreva seu bot campeão no Torneio. Disputa em fases com grupos de 8 programas, classificação automática e critérios de desempate claros.

Assista aos replays das partidas já processadas, analise os  logs  e compare suas decisões com as dos oponentes para evoluir.
